# Mockup Reference — SheetSight

Acuan visual untuk membangun SheetSight.

## Cara pakai
Buka **`SheetSight-mockup.html`** di browser untuk melihat arah desain lengkap:
hero, upload card, layout dashboard (KPI cards, panel grafik, tabel), warna, tipografi, dan spacing.

> Ini **mockup statis** (tampilan saja) — bukan aplikasi fungsional. Tujuannya menunjukkan *seperti apa hasil akhirnya*. Logika upload/parsing/grafik dibangun sesuai `PRD.md`.

## Ambil token visual dari mockup
- Dark default: bg `#0A0E1A`, surface `#151B2B`
- Aksen indigo `#4F46E5` → `#8B8CFB`, aksen2 cyan `#06B6D4`
- Font: Source Sans Pro (UI) + JetBrains Mono (angka/label)
- Radius kartu 14–20px, mobile-first

Jangan tiru pixel-perfect — tangkap *rasa* dan sistem desainnya, lalu terapkan ke komponen React fungsional.
